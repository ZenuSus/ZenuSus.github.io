Скриншоты

Как сыграть

1. Скачайте zip-архив shaverma.zip
2. Разархивируйте его в любое удобное вам место
3. Скачайте python с python.org
4. Откройте терминал/консоль, и установите зависимости через pip (pip install pygame)
5. Запустите игру ( python shaverma.py)
6. Готово!
